# Issues & Troubleshooting

본 문서는 개발 및 운영 과정에서 발생한 주요 이슈와  
그에 대한 **원인 분석 및 해결 과정을 정리**한 기록입니다.

---

## Issue 1. 상세 화면 진입 시 데이터가 정상적으로 표시되지 않음

### 상황
- 목록 페이지에서 특정 항목 선택 후 상세 화면 이동
- API 응답은 정상이나 화면에 데이터가 표시되지 않음

---

### 원인 분석
- 상세 페이지 진입 시점에 상태 값이 아직 세팅되지 않은 상태
- 컴포넌트 렌더링 타이밍과 데이터 할당 시점 불일치

---

### 해결 방법
- `onMounted` 시점에서 데이터 로딩 로직 재구성
- 상태 값이 준비된 이후에 렌더링되도록 조건 처리

---

### 결과
- 상세 화면 정상 렌더링
- 동일한 패턴의 화면에서도 재사용 가능하도록 구조 개선

---

### 배운 점
- Vue에서 상태 관리 시 **렌더링 타이밍**이 중요함
- 단순 API 문제로 보이는 이슈도 프론트 구조 문제일 수 있음

---

## Issue 2. 권한에 따라 화면이 잘못 노출되는 문제

### 상황
- 관리자 전용 기능이 일반 사용자에게 노출됨

---

### 원인 분석
- 권한 체크 로직이 일부 컴포넌트에만 적용됨
- 공통 가드 부재

---

### 해결 방법
- 권한 기반 공통 처리 로직 구성
- UI 렌더링 시 권한 조건 명확화

---

### 결과
- 권한별 UI 안정성 확보
- 유지보수 용이성 향상

---

### 배운 점
- 권한 로직은 **중앙화**하는 것이 중요
- UI 단에서도 보안 관점이 필요함
