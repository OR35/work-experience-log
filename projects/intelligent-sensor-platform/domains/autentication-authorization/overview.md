# Authentication / Autorization Domain

## 📌 Overview

사용자와 관리자의 접근을 제어하기 위해  
JWT 기반 인증/인가 구조를 적용한 도메인입니다.

본 프로젝트에서는  
로그인 이후 발급된 토큰을 기준으로 사용자 상태를 유지하고,  
권한에 따라 접근 가능한 화면과 API를 분리하도록 구성했습니다.

인증 상태는 프론트엔드와 백엔드 양쪽에서 처리되며,  
저는 인증 흐름에 필요한 화면, API 연동, 권한 분기 로직을 구현했습니다.

---

## 🔧 Implemented Features

- 로그인 화면 구현 및 인증 API 연동
- 로그인 성공 시 JWT 발급 및 저장 처리
- 인증 상태에 따른 화면 접근 제어
- 사용자 / 관리자 권한별 메뉴 및 기능 분기
- 토큰 만료 시 재로그인 유도 처리
