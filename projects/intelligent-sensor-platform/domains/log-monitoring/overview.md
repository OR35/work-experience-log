# Log & Monitoring Domain

## 📌 Overview

플랫폼 운영 과정에서 발생하는  
접근, 이벤트, 발송, 작업 이력 및 시스템 로그를 관리하는 도메인입니다.

운영자가 서비스 상태를 빠르게 파악할 수 있도록  
목록 / 상세 조회 중심의 화면 흐름을 구성했습니다.

저는 로그 데이터 조회를 위한 화면 구성과  
상태 확인에 필요한 정보 가공 및 표시 로직을 구현했습니다.

---

## 🔧 Implemented Features

- 접근 / 발송 / 이벤트 / 작업 이력 관리
- 시스템 로그 조회
- 운영 이슈 추적 기능
