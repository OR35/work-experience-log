# Profile

> 실무에서 **직접 담당하고 구현한 기능과 문제 해결 과정**을 중심으로  
> 플랫폼 서비스 개발 경험을 쌓아온 개발자입니다.

---

## 💡 Overview

플랫폼 서비스 환경에서
**Frontend와 Backend 기능을 개발**하며,사용자 중심의 화면과  
안정적인 비즈니스 로직 구현 경험을 쌓아왔습니다.

- 요구사항을 기능 단위로 구조화하여 구현
- 사용자/관리자 권한 분리 및 상태 관리
- 운영 환경을 고려한 화면 흐름과 데이터 처리

경험을 프로젝트 단위로 정리하고 있습니다.

---

## 🛠 Core Capabilities

### Backend
- Java, Spring Boot, Spring Security
- JPA 기반 데이터 모델링
- JWT 기반 인증 / 인가 처리
- 화면 요구사항에 맞춘 API 응답 설계 및 데이터 가공

### Frontend
- Vue.js, JavaScript
- CRUD 기반 화면 구현 (목록 / 상세 / 등록 / 수정 / 삭제)
- 사용자 / 관리자 권한에 따른 UI 분기
- Tailwind CSS, CodeMirror 기반 에디터 화면 구성

### Platform Integration
- 외부 시스템 API 연동을 통한 기능 구현
- 모델 학습 / 최적화 Job 실행 API 연계
- Job 상태 추적 및 결과 데이터 화면 반영
- 로그 기반 결과 데이터 정합성 관리

### Development Productivity
- ESLint / Prettier 기반 코드 컨벤션 정리
- Husky + Lint-staged를 활용한 Commit 단계 품질 관리
- 팀 협업 시 코드 스타일 및 유지보수성 개선 경험

복잡한 알고리즘이나 과도한 기술 적용보다는,

- 운영 중 발생하는 예외 상황
